# **App Name**: JobHub Enhanced

## Core Features:

- Dark Mode Toggle: Enables users to switch between light and dark themes for optimal viewing experience.
- Theme Persistence: Saves the user's theme preference in local storage to maintain consistency across sessions.
- Inter Font Integration: Applies the Inter font across the application for improved readability and modern aesthetics.
- Dynamic Styling Adjustment: The application automatically adjusts its styling based on the chosen theme, ensuring that all components are visually consistent and accessible.
- Context Provider: Utilize the ThemeContext to manage and provide theme data throughout the application.
- Theme transition animation: A smooth, animated transition when the user toggles between light and dark mode.

## Style Guidelines:

- Primary color: Blue (#3B82F6) for main interactive elements.
- Background color: Light gray (#F9FAFB) for light mode and dark gray (#1F2937) for dark mode.
- Accent color: Teal (#14B8A6) for highlights and active states.
- Font: 'Inter', a sans-serif font, is used for headlines and body text for its modern and readable appearance.
- Use simple, outlined icons from React Icons for visual clarity and consistency across the platform.
- Employ a clean, modern layout with ample white space to enhance readability and reduce visual clutter.
- Implement subtle transitions and animations to provide a smooth user experience when navigating and interacting with the app.
